# ElimuSmartPlan.STEM

AI-powered Smart Lesson Planner for STEM teachers — built with FastAPI, Jac Language, ByLLM, and React.

##  Features
- Generate lesson plans using AI (ByLLM + Google Generative AI)
- Use Jac language to define logic and flow
- React + TailwindCSS modern interface
- Dockerized for easy deployment


## Tech Stack
- **Frontend:** React, TailwindCSS, Vite
- **Backend:** FastAPI, Jaseci, JacLang, ByLLM
- **Database:** Redis (cache)
- **Deployment:** Docker & Docker Compose
